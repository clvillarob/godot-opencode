# Godot Opencode

Plugin para Godot 4.6+ que integra el agente de IA [opencode](https://opencode.ai) directamente en el editor de Godot. Chatea con IA, explora archivos y ejecuta comandos sin salir del editor.

## Características

- **Chat interactivo:** Conversación con opencode con contexto del editor
- **Comandos predefinidos:** Explicar, refactorizar, mejorar y documentar código con un clic
- **Explorador de archivos:** Navega tu proyecto Godot desde un dock nativo
- **Consola integrada:** Salida de opencode y línea de comandos en tiempo real
- **EditorBridge:** Aplica automáticamente los cambios generados por opencode

## Instalación

1. Copia `addons/godot-opencode/` en la carpeta `addons/` de tu proyecto
2. Actívalo en **Proyecto > Configuración del proyecto > Plugins**
3. Asegúrate de tener [opencode](https://opencode.ai) instalado y configurado
